## :computer: Ecoleta

Ecoleta - é uma forma de conectar empresas e entidades de coleta de resíduos orgânicos e inorgânicos as pessoas que precisam descartar seus resíduos de maneira ecológica.

## :rocket: Tecnologias

- Typescript
- React
